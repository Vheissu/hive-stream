# Operation Builder Examples

These examples show the chainable `streamer.ops` layer for common write operations.

- `transfer-builder.ts` - Build and send a HIVE/HBD transfer with chained methods
- `proposal-builder.ts` - Build and send a DHF proposal with chained methods

The same namespace also exposes chainable builders for:

- Hive Engine transfers, burns, and issuance
- Proposal vote updates and proposal removal
- Upvotes and downvotes using the configured voter
